This is a [Next.js](https://nextjs.org/) and [FastAPI](https://fastapi.tiangolo.com/) project.

## Getting Started

First, run the development server:
```bash
uvicorn main:app
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
